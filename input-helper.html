Input Figure Naming Guide

Place files under templates/inputs or your selected inputs folder used by the report.

Manual Profile Input

  • <profile_name>.png
  • <profile_name>-wp.png
  • <profile_name>p.png
  • <profile_name>-lb-web.png
  • <profile_name>-lb-flange.png
  • <profile_name>-lb-table.png
  • <profile_name>-lb-web-cap.png
  • <profile_name>-lb-flange-cap.png

Wind Input

  • location-map.png
  • mwfrs.png (MWFRS load table)
  • cnc-wall.png
  • cnc-roof.png

Category-level Figures

  • Reference elevation: <category_index>-ref-elev.png (e.g., 1-ref-elev.png)
  • SAP/structural (per category index):
    • <category_index>-sap-model.png
    • <category_index>-sap-relese.png
    • <category_index>-sap-dead-load.png
    • <category_index>-sap-wind-load.png
    • <category_index>-sap-bmd.png
    • <category_index>-sap-sfd.png
    • <category_index>-sap-mul-max-moment.png
    • <category_index>-sap-tran-max-moment.png
    • <category_index>-sap-deformed-shape.png
    • <category_index>-sap-mul-def.png
    • <category_index>-sap-tran-def-wind.png
    • <category_index>-sap-tran-def-dead.png
    • <category_index>-sap-joint-force-dead.png
    • <category_index>-sap-joint-force-wind.png
    • <category_index>-sap-reaction-force-dead.png
    • <category_index>-sap-reaction-force-wind.png

Glass RFEM Figures (category + glass indices)

Format: <category_index>.<glass_index>-rfem-<suffix>.png (e.g., 2.3-rfem-model-3d.png)

  • <c>.<g>-rfem-model-3d.png
  • <c>.<g>-rfem-model-data.png
  • <c>.<g>-rfem-stress.png
  • <c>.<g>-rfem-stress-ratio.png
  • <c>.<g>-rfem-def.png
  • <c>.<g>-rfem-def-ratio.png

Notes

  • category_index is 1-based from the order of categories in the input YAML.
  • glass_index is 1-based within each category.
  • Use lowercase .png extensions to match template references.
